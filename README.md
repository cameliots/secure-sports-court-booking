# Secure Sports Court Booking System
The Secure Sports Court Booking System is a Django-based web application developed to manage sports court reservations securely. Users can view available courts and create bookings, while administrators manage courts and bookings through the Django admin interface.

## Technologies Used
- Python 3
- Django 4.2
- SQLite
- HTML, CSS
- Git & GitHub

## System Features
- User authentication and authorization
- Court booking management
- Admin dashboard for system management
- Secure configuration using environment variables
